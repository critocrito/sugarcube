### Installation

    npm install --save @sugarcube/plugin-twitter


### Usage

Fetch individual tweets by the Tweet ID.

**Configuration:**

-   `twitter.consumer_key`
-   `twitter.consumer_secret`
-   `twitter.access_token_key`
-   `twitter.access_token_secret`
